巴黎一日游 & 现场看奥运

12 Aug, 2024

前往巴黎

巴黎离苏黎世说近不近，说远也不算远。这次巴黎奥运机会难得，于是就兴起买票现场体验一下奥运会的氛围，顺便也在巴黎来一个一日游的city walk。

看奥运当然希望看到中国队的比赛，于是就买了乒乓球决赛的门票，在比赛开始前我就预测到了大概是中国和日本的对决，最后果不其然也是这个结果。我即看到了中国的夺冠时刻，在现场升国旗、唱国歌，也看到了日本乒乓美女小苹果（早田希娜），还是很满意的！

这次去巴黎我是做夜大巴前一天晚上出发，第二天晚上看完比赛就直接返回苏黎世。其实最早计划是在巴黎住一晚，就连往返机票都买好了。但我查了一下奥运会期间酒店的价格——实在是涨的太离谱了，于是就取消了机票，改成了大巴。

从苏黎世到巴黎的夜大巴要做9个半小时，晚上21:00从苏黎世出发，第二天早上6点半到巴黎。我是第一次在欧洲坐这么长时间的巴士。之前在小红书上看到很多劝退的帖子，说长时间做大巴屁股疼，而且车里的卫生也很差。但我实际乘坐下来感觉其实没有那么糟糕，巴士的椅子大概能后倾到120度左右，感觉比国内的高铁二等座要舒服。车里面的卫生感觉也还可以——我没用（也没有胆量）去体验车里的厕所，但是车厢里面是没有异味的。在去巴黎的车上，我一直在刷手机看视频一直到凌晨两三点，最后实在困了才眯了一会。偶对了，这里有一个小插曲，就是自己在大巴上收到了自己的第一篇一作文章被BioCAS接受了！得知文章接收的消息之后，心情非常舒畅！迷迷糊糊睡了几个小时，第二天早上六点刚过一点点，大巴就提前到了巴黎。

马拉松比赛现场

买了一张交通天票后，就立刻做地铁到达第一站——巴黎圣母院。没想到在去巴黎圣母院的路上碰到了即将开始的奥运男子马拉松比赛。比赛早上9点开始，还不到时间，于是我在路旁边占了一个好位置等待。随着比赛开始的临近，现场的气氛也越来越浓。很多拿着各国国旗的观众在助威呐喊，我也看到了几个中国的马拉松爱好者来观赛，他们也都举着中国国旗助威。当然，我还看到很多专业的摄影设备，毕竟是奥运会比赛——是要全球直播的，我还在像自己运气好的话说不定可以被直播找到（哈哈，后来我看了回放自己并没有被照到，但站位就差一点点）。早上九点，随着一声枪响，比赛正式开始，各个国家的马拉松选手应声出发，现场的气氛也到达了高潮。各种助威的声音，让我切实体会了奥运会的氛围。

打卡巴黎著名景点

看完马拉松气泡，我就按着之前在大巴上计划好的路线，开始了巴黎的city walk。主要路线是：巴黎圣母院 => 卢浮宫 => 协和广场 => 香榭丽大街 => 凯旋门 => 埃菲尔铁塔。而且因为这次时间比较紧张，我就只在各个景点前面打了一下卡，都没有进去逛。

感觉自己在奥运期间来真的是很幸运，很多景点都是“奥运限定”。比如卢浮宫玻璃金字塔前面的奥运五环，杜乐丽公园里面热气球一样的奥运圣火，凯旋门和巴黎铁塔上悬挂的奥运会、残奥运会徽等等——无处不在的奥运会气氛

巴黎上一次举办奥运会还是100年前，错过这次奥运真不知道下一次会在什么时候。
