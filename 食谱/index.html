调味类

油泼辣子

食材：辣椒面、白芝麻、盐、五香粉

  1. 加辣椒面、白芝麻、盐、五香粉
  2. 泼油搅拌

蒜水

食材：辣椒面、蒜、醋

  1. 套蒜末
  2. 加辣椒面、盐
  3. 泼热油
  4. 加适量醋

小吃类

鸡蛋早餐饼

食材：鸡蛋、包菜、（虾仁）；淀粉、盐、黑胡椒、葱

  1. 虾仁去皮，加淀粉黑胡椒腌制
  2. 鸡蛋、盐、黑胡椒、葱花搅拌
  3. 锅底喷油加入虾仁煎熟
  4. 加入蛋液定形、反面将饼煎熟
  5. 出锅！

面食

油泼面

食材：葱花、蒜、小青菜；西北辣椒面、五香粉、酱油、醋

  1. 葱、姜切末
  2. 面煮熟、小青菜烫熟，放碗里备用
  3. 面上加葱末、蒜末、辣椒面、五香粉、盐少许
  4. 热油泼到面上，搅拌
  5. 加适量酱油和醋

鸡蛋高汤面

食材：鸡蛋、蘑菇、大葱；白胡椒

  1. 葱切段，葱叶切葱花，蘑菇切片备用
  2. 面条煮熟备用
  3. 起锅热油，煎两个鸡蛋，煎至蛋黄溏心
  4. 加入开水，加入葱段、蘑菇，加入适量盐、白胡椒，大火煮5-10分钟至汤汁浓稠
  5. 加入面条，小火再煮一小会
  6. 撒上葱花，出锅！

米饭

咖喱鸡肉饭

食材：鸡腿肉、胡萝卜、土豆、洋葱、咖喱块

  1. 鸡腿肉切小块，加盐、黑胡椒抓匀腌制15分钟
  2. 胡萝卜、土豆切丁，洋葱切好备用
  3. 热油，加洋葱炒软；再加胡萝卜土豆丁，鸡腿肉至变色
  4. 加开水，放咖喱块，煮至浓稠

家庭小炒

花菜炒鸡蛋

食材：花菜、鸡蛋、虾仁、胡萝卜、小葱；生抽

  1. 花菜切碎备用
  2. 锅里喷油，打入鸡蛋搅碎，加入花菜碎、虾仁、胡萝卜丁，加入盐、生抽调味，翻炒熟透
  3. 出锅前加入小葱花提香

西蓝花炒口菇

食材：西蓝花、口菇、胡萝卜、蒜；生抽

  1. 西蓝花、口菇、胡萝卜全部焯水备用
  2. 蒜末炒香，加入蔬菜入锅翻炒，加一点生抽调味

手撕包菜

食材：包菜、干辣椒、葱、蒜；五香粉、醋、糖、酱油、鸡精

  1. 手撕包菜备用
  2. 起锅热油，加干辣椒、葱、蒜丁炒香
  3. 加入包菜炒软
  4. 加五香粉、醋、糖、盐、酱油翻炒均匀
  5. 加少许鸡精提鲜

拌饭、面臊子

土豆火腿

食材：土豆、洋葱、火腿；生抽

  1. 土豆切片，加油炒熟盛起备用
  2. 洋葱炒香，加火腿翻炒
  3. 加入土豆片，加生抽翻炒均匀

猪肉臊子

食材：葱、姜、猪肉馅(~400g)；酱油、料酒、糖、豆瓣酱

酱汁：150ml 水，3大勺酱油，3大勺料酒，4小勺糖

  1. 姜和蒜切成沫
  2. 中火起锅烧油（~4大勺芝麻油），下入葱姜沫炒香
  3. 下入猪肉馅，翻炒直至完全变色
  4. 倒入酱汁，加入4小勺豆瓣酱
  5. 大火收汁，出锅

土豆烧鸡翅

食材：鸡翅、姜、蒜、土豆、小米辣椒；八角、黄豆酱、生抽、耗油

  1. 鸡翅对半剪开，加入姜片、料酒、清水，将鸡翅煮熟备用
  2. 热油，加入鸡翅煮至两面金黄
  3. 加入大蒜、八角、黄豆酱、生抽、耗油
  4. 加入土豆块和清水，小伙焖煮20分钟，加点小米辣椒
